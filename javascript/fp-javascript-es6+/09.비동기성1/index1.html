# Promise - Promise라는 값은 대기, 성공, 실패를 다루는 1급 값으로 이루어져 있음. callback과 차이점이 있음. - 비동기 상황을 1급 값으로 다룸. - 비동기 상황에 대한 값을 return한다는 게 중요함. ## 일급 ## 일급 활용 - 비동기 상황을 값으로 다룰 수 있다는 것은 1급의 성질을 갖고 있다는 것 - 비동기 상황이 값이라는 것은 함수에 전달할 수 있음. ## Composition - Promise는 비동기 상황에서 함수 합성을 안전하게 해주는 도구. - 비동기 값을 가지고 연속적인 함수 실행을 안전하게 해주는(함수 합성을 안전하게 하는) Monad라고도 할 수 있음. - Monad - 함수 합성
